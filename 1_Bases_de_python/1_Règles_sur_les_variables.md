
# Règles sur les variables en Python

- Une variable commence avec une **lettre**
- Une variable ne commence pas avec un  nombre
- Pas d'espaces dans le nom des variables ou de caractères spéciaux
- On peut utiliser un **underscore** (_) pour remplacer l'espace si on utilise la convention SnakeCase
- Il faut éviter les mots clés du langage tels que ```ìf```, ```else```,```while```, ```class```, ```print``` etc ...
- Il faut donner des noms explicites/clairs au variable
- Les noms de variable respectent la casse
- Pour voir le **type** d'une variable, utilises la fonction ```type()```, penses à l'utiliser dans ```print()```
