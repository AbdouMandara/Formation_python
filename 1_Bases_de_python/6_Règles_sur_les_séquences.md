# A savoir sur "les séquences"

La **Séquence** est un ensemble fini et ordonné d'élement indicés de 0 à n-1 (n étant le nombre d'élements de cette séquence)

La séquence peut etre : **Une chaine de caractères**, **une liste**, **un tuple**, **un set** etc ...
Nb: Tous les élements **conteneurs** sont des ````séquences```

Pour accéder à un élément d’indice i d’une séquence, on utilise la syntaxe : ```nom_de_la_sequence[i]```

## Test d'appartenance

Le test d’appartenance est une opération qui permet de vérifier si un élément fait ou pas parti d’une séquence
 Pour voir si un élement appartient à une séquence on utilise le **in** pour voir s'il n'est pas présen **not**
