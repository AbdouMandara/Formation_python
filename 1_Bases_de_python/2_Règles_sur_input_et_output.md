# Utilisation de input() et output()

## Utilisation de input()

On utilise ```ìnput()``` pour demander au user d'entrer une valeur par défaut on récupère une valeur de type ```str```

Pour convertir ce que le user a entré à travers ```ìnput()```  nous mettons ce input dans ```ìnt()```

## Utilisation de output()

On utilise ```print()``` pour afficher un message à l'écran
