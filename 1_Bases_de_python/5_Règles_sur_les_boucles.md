# Les boucles

En python, on distingue les boucles : **While** et **for**

## Boucle "While"

Pour éxecuter un bloc d'instructions, lorsqu'une condition est vraie

## Boucle "For"

On l'utilise principalement lorsqu'on veut parcourir un sequenceur ou structure de données (listes, set, tuples)
Quand j'utilise **range()**, par exemple range(3) va créer une séquence virtuelle de nombre entiers allant de 0 à 2 donc (0, 1 et 2)

```Continue``` dans une boucle **for** veut dire : **Tout ce qui se trouve après lui est ignoré** et on passe à l'itération suivante dans la boucle si la condition est encore **vrai** pour exécuter le bout de code
```Break``` dans une boucle **for** veut dire : permet de sortir complètement de celle-ci ; même si
elle n’avait pas terminé son exécution.
