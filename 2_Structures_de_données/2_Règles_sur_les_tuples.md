# Les Tuples

Ils sont comme les **listes** mais ils sont **immuables** donc on peut pas les modifier comme les listes
**Syntaxe** : nomDuTuple = (elt1, elt2, ..., eltN) ou nomDuTuple = tuple()

**Utilité des tuples** : on peut les utiliser pour stocker des données fixes pour économiser de l'espace mémoire
