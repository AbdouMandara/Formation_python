# Lancement / Execution d'un fichier en python

 ````py test1.py```
ou
````python test1.py```

## Savoirs

- Python est un langage **dynamique**
- Pour commenter on a : **commentaires multilignes** et **commentaires monolignes**
- En Python, pas besoin de terminer une instruction ou ligne avec un point virgule (;) comme en PHP, Java, C etc...
